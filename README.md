## About Me

I'm a **BS Computer Science student** who learns by building, breaking, and debugging things until they make sense. I care about writing practical software on strong fundamentals — data structures, algorithms, OOP, and clean architecture — and I only claim a skill once I've genuinely learned it.

- **Full-Stack Development** — building web applications with Node.js and PostgreSQL, from database schema to UI
- **AI / ML & Autonomous Systems** — hands-on computer vision work with PyTorch, YOLOv8, and real-time inference pipelines
- **Systems & Low-Level** — C, C++, and x86 Assembly projects that go under the hood
- **Product Mindset** — every project ships as something usable, not just a code dump

**Open to:** open-source collaboration, internships, and building alongside people who take engineering seriously.
